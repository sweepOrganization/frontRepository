# 지각하지마 미리미리

도착 시간을 기준으로 출발 시간을 계산하고, 이동 전에 알림을 보내 주는 프론트엔드 프로젝트입니다.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- TanStack Query
- Zustand

## 시작하기

```bash
npm install
npm run dev
```

## 주요 스크립트

| 명령              | 설명               |
| ----------------- | ------------------ |
| `npm run dev`     | 개발 서버 실행     |
| `npm run build`   | 프로덕션 빌드      |
| `npm run preview` | 빌드 결과 미리보기 |
| `npm run lint`    | ESLint 실행        |
