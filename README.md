## 시작하기

```bash
npm install
npm run dev
```

### 주요 스크립트

| 명령                      | 설명                  |
| ------------------------- | --------------------- |
| `npm run dev`             | 개발 서버 실행        |
| `npm run build`           | 프로덕션 빌드         |
| `npm run preview`         | 빌드 결과 미리보기    |
| `npm run lint`            | ESLint 실행           |
| `npm run storybook`       | Storybook (포트 6006) |
| `npm run build-storybook` | Storybook 정적 빌드   |
